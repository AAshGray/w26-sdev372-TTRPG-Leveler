# TTRPG-Leveler
## 1. Project Overview
Solve the issue of crappy character sheets past level 1 for TTRPG
players who would prefer a more streamlined and automated way to
level up their characters.
## 2. Feature Breakdown
### MVP Features
- DnD Character sheet that can, when the user interacts with it,
is able to update stats abilities or what exact options they have
at their disposal.
